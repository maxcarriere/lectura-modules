1. **GNU Affero General Public License v3.0 (AGPL-3.0)** — gratuite,
   avec obligation de publication du code source de tout logiciel derivé.
2. **Licence Commerciale Lectura** — payante, sans obligation de

Ce document decrit les termes de la **Licence Commerciale**.

## 1. Objet

La Licence Commerciale Lectura accorde au Licencié le droit d'utiliser,
modifier et integrer les modules Lectura NLP dans des logiciels
proprietaires, sans etre soumis aux obligations de l'AGPL-3.0.

## 2. Droits accordes

Sous reserve du paiement de la redevance applicable, le Licencié obtient
un droit non exclusif et non transférable de :

- **Utiliser** les modules dans un nombre illimité de projets internes.
- **Modifier** le code source pour l'adapter à ses besoins.
- **Integrer** les modules dans des logiciels proprietaires distribues
  à des tiers, sans obligation de publier le code source.
- **Distribuer** les modules sous forme compilée au sein de ses produits.

## 3. Restrictions

- Le Licencié **ne peut pas** redistribuer le code source des modules
  en tant que produit autonome ou bibliotheque independante.
- Le Licencié **ne peut pas** sous-licencier les modules à des tiers
  en dehors de l'integration dans ses propres produits.
- Le Licencié **doit** maintenir les notices de copyright dans le code.

## 4. Modeles pre-entraines

Les fichiers de modeles (.onnx, .json de poids) inclus dans les modules
sont couverts par cette licence commerciale. Leur utilisation en dehors
des modules Lectura NLP nécessite un accord séparé.

## 5. Support et mises à jour

- Le support technique par email est inclus.
- Le support prioritaire et le consulting sont disponibles séparément.
